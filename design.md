# Функциональные требования к приложению

## Главный экран

> Используемые юзкейсы - get_user_info, select_cards_to_fixation, select_cards_to_lesson.

Центральный хаб для отслеживания прогресса и быстрого доступа к обучению.

Персонализация: Приветствие пользователя и отображение аватара профиля.

Кнопка Урок, кнопка Закрепление.

Блок «Overview» (Статистика):

* Total Cards: Общее количество карточек в базе с индикатором прироста за неделю.
* Learned: Количество изученных карточек со статусом «Mastered».
* In Progress: Карточки в процессе изучения, требующие повторения («Needs review»).
* New: Новые карточки, к которым пользователь еще не приступал.
* Сложные слова: Отдельный счетчик слов с низким процентом правильных ответов.

Каждая карточка статистики содержит кнопку «История» с графиком для детального анализа динамики.

Нижняя навигация (Tab Bar): Быстрый переход между разделами: Главная, Слова, Кандзи, Грамматика.

## Библиотеки контента (Слова, Кандзи, Грамматика)

Экраны для просмотра и управления учебным материалом.

### Слова

> Используемые юзкейсы - knowledge_set_cards, create_vocabulary_card, delete_card

Перечень слов, которые пользователь изучает.

Функции:

* Поиск по ключевым словам/переводу.
* Кнопка добавления нового слова (+).
* Список карточек с отображением перевода, даты следующего повторения, метрика сложности, метрика стабильности.
* Теги категорий - Новое, Сложное, В процессе изучения, Изученные.

### Кандзи

> Используемые юзкейсы - knowledge_set_cards, create_kanji_card, delete_card, kanji_info, kanji_list.

Перечень кандзи японского языка, группировка по уровням JLPT (N5–N1).

Функции:

* Поиск по кандзи/переводу.
* Кнопка взять кандзи на изучение (+).
* Список карточек с отображением кандзи, значения, произношения, даты следующего повторения, метрика сложности, метрика стабильности, радикалы кандзи.
* Теги категорий - Новое, Сложное, В процессе изучения, Изученные.

### Грамматика

> Используемые юзкейсы - knowledge_set_cards, create_grammar_card, delete_card, grammar_info, grammar_list.

Перечень грамматических конструкций японского языка, группировка по уровням JLPT (N5–N1).

Функции:

* Поиск по грамматическим конструкциям.
* Кнопка взять грамматическую конструкцию на изучение (+).
* Список карточек с отображением грамматической конструкции, даты следующего повторения, метрика сложности, метрика стабильности.
* Теги категорий - Новое, Сложное, В процессе изучения, Изученные.

## Процесс обучения

> Используемые юзкейсы - complete_lesson, select_cards_to_fixation, select_cards_to_lesson, rate_card.

Интерфейс режима активного запоминания.

Прогресс-бар: Отображение текущей позиции в уроке (например, 12 / 42) и визуальная шкала.

Карточка для обучения:

* Тип карточки: Кандзи, Слово или Грамматическая конструкция.
* Вопрос - Слово\Кандзи\Название грамматической конструкции.
* Кнопка прослушивания аудио-произношения (только для слов).

4 кнопки для оценки:

* Не знаю
* Плохо
* Знаю
* Идеально

Обратная сторона карточки (показывается после оценки)

* Для слов: Перевод.
* Для кандзи: Значения, все варианты, радикалы.
* Для грамматики: Подробное объяснение конструкции, правила присоединения к частям речи.

Кнопка «Далее»: Для перехода к следующей карточке после ознакомления с деталями.

## Профиль пользователя

Настройки и интеграции.

* Личные данные: имя пользователя.
* Выбор целевого уровня JLPT.
* Интеграция с Duolingo: Возможность подключения стороннего аккаунта для синхронизации или импорта данных.
* Настройки приложения: Выбор языка интерфейса (например, Русский).
* Кнопка выхода из аккаунта.

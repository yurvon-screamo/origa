# Origa — Возможности для пользователя

Origa — приложение для изучения японского языка. Ниже описано, что оно умеет делать.

---

## Интервальные повторения (FSRS)

Приложение использует алгоритм FSRS для планирования повторений.
Это значит, что карточки появляются в оптимальный момент
для запоминания — не слишком рано (забыл), не слишком поздно (забыл).

Четыре варианта ответа на карточку:

| Ответ | Когда использовать |
| ------ | ------------------- |
| Again | Неверный ответ |
| Hard | Правильный, но с затруднением |
| Good | Правильный, без проблем |
| Easy | Правильный, слишком легко |

Интервалы подбираются автоматически: от нескольких минут после
Again до месяцев и более после нескольких Easy подряд.

---

## Типы карточек

**Слова** — слово на японском, чтение (фуригана), перевод. Пример: 猫 (ねこ) — кошка.

**Кандзи** — иероглиф с чтениями (on'yomi, kun'yomi), списком
радикалов, примерами слов. Для кандзи доступны прописи —
рисование пальцем или мышью с автоматической проверкой.

**Грамматика** — грамматическая конструкция с объяснением и примерами использования.

Четыре формата практики:

- **Вспоминание** — проверить, помните ли вы значение, чтение
  или перевод (кнопки Again / Hard / Good / Easy)
- **Да/Нет** — подтвердить или опровергнуть понимание
- **Викторина** — выбрать правильный ответ из предложенных
- **Прописи** — нарисовать кандзи с автоматической проверкой

## Пополнение словаря

**Вручную** — ввести слово. Приложение само найдёт перевод из встроенного словаря.

**OCR** — сфотографировать текст. Приложение распознает японские
символы на изображении, приводит слова к базовой форме и
автоматически находит перевод из встроенного словаря. Останется
только создать карточки для нужных слов.

**Импорт готовых наборов** — загрузить коллекцию карточек,
созданную другими пользователями или в других приложениях.

**Импорт из Anki** — загрузить колоду в формате `.apkg`.
Приложение автоматически определит поля (слово, перевод),
покажет превью карточек и импортирует их в ваш словарь.
Работает в браузере и в десктопном приложении.

## Автоматическая фуригана

Фуригана — это чтение над кандзи. Приложение добавляет её
автоматически поверх любого текста.

Если кандзи в вашем словаре — фуригана скрыта (вы и так знаете
чтение). Если не знаете — чтение показано.

Работает в карточках, в объяснениях, в любом тексте приложения.

---

## Отслеживание прогресса по JLPT

JLPT — стандартизированный экзамен по японскому.
Пять уровней: N5 (начальный) → N1 (продвинутый).

Приложение отслеживает, сколько кандзи, слов и грамматики вы выучили на каждом уровне.

| Уровень | Кандзи |
| -------- | ------ |
| N5 | 80 |
| N4 | 166 |
| N3 | 367 |
| N2 | 367 |
| N1 | 194 |

Когда достигаете 90% прогресса на уровне — приложение предложит перейти к следующему.

## Избранные карточки

Любую карточку можно добавить в избранное. Избранные карточки
появляются в уроках в первую очередь, независимо от расписания.

Если карточку в избранном ответить Easy пять раз подряд —
она автоматически удаляется из избранного (считается усвоенной).

---

## Анализ текста

Вставьте текст на японском языке — приложение разберёт его и покажет:

- какие слова из текста вы уже знаете
- какие слова новые (можно создать карточки)
- какие кандзи знаете / не знаете

Это позволяет превратить любой текст — статью, книгу, пост — в учебный материал.

---

## Персонализация

**Темп обучения** — сколько новых карточек добавлять в день. Настраивается в профиле.

**Родной язык** — влияет на отображение переводов.

**Онбординг** — при первом запуске: выбор текущего уровня
(или что уже знаете), указание источников предыдущего обучения
(если были), оценка начальных знаний. На основе этого
подбираются стартовые наборы карточек.

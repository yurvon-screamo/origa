# User Journey: Learning - Lesson

## Цель

Пользователь проходит урок: система отбирает карточки для изучения, пользователь оценивает каждую.

---

## Сценарий

### Step 1: Выбор карточек для урока

**Given:** У пользователя есть карточки в knowledge_set

**When:** Запрашиваются карточки для урока

**Then:**
- Возвращается map карточек для урока
- Приоритет: favorite → high_difficulty + due → new → known + due
- Лимит: 7 новых карточек + известные с due date
- Карточки перемешаны
- Каждая карточка представлена в одном из view: Normal, Quiz, Reversed, GrammarMutated

---

### Step 2: Оценка карточки (Rating::Again)

**Given:** Карточка показана пользователю

**When:** Пользователь не вспомнил (Again)

**Then:**
- Memory state обновлён: stability уменьшена
- Difficulty увеличена
- Карточка появится в ближайшем закреплении

---

### Step 3: Оценка карточки (Rating::Hard)

**Given:** Карточка показана пользователю

**When:** Пользователь вспомнил с трудом (Hard)

**Then:**
- Memory state обновлён: stability немного увеличена
- Difficulty немного увеличена

---

### Step 4: Оценка карточки (Rating::Good)

**Given:** Карточка показана пользователю

**When:** Пользователь вспомнил нормально (Good)

**Then:**
- Memory state обновлён: stability увеличена
- Difficulty не изменена или уменьшена
- Интервал до следующего повторения увеличен

---

### Step 5: Оценка карточки (Rating::Easy)

**Given:** Карточка показана пользователю

**When:** Пользователь вспомнил легко (Easy)

**Then:**
- Memory state обновлён: stability значительно увеличена
- Difficulty уменьшена
- Интервал до следующего повторения значительно увеличен

---

### Step 6: Полный цикл урока

**Given:** Пользователь начинает урок

**When:** Проходит через все отобранные карточки с оценками

**Then:**
- Все карточки оценены
- Lesson history обновлён (lessons_completed++)
- Daily stats обновлены

---

## View Types

| View | Описание | Когда применяется |
|------|----------|-------------------|
| Normal | Вопрос → Ответ | Все типы карточек |
| Quiz | Вопрос → 4 варианта ответа | Vocab, Kanji (новые и известные) |
| Reversed | Ответ → Вопрос | Vocab (только известные) |
| GrammarMutated | Слово с применённым правилом | Vocab (только известные, если есть grammar cards) |

---

## Edge Cases

### EmptyLesson - нет карточек
- **Given:** Knowledge_set пуст
- **When:** Запрашиваются карточки для урока
- **Then:** Возвращается пустой map

### CardNotFound при RateCard
- **Given:** Оценивается карточка
- **When:** card_id не существует
- **Then:** Ошибка CardNotFound

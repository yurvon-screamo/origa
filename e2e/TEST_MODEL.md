# E2E Test Model v3

## Overview

Документ описывает end-to-end тестирование приложения Origa на основе **реальных пользовательских сценариев** (User Journeys).

## Принципы

1. **Тесты создают данные** — каждый тест самостоятельно создаёт контент для проверки
2. **Изоляция** — тесты независимы и не влияют друг на друга
3. **Реальный путь** — тестируем то, как реально пользуется приложением человек

---

## User Journey 1: Полный цикл обучения (Primary)

**Цель:** Проверить основной сценарий использования приложения — от входа до завершения урока.

### Предусловия

- Зарегистрированный тестовый пользователь (email подтверждён)

### Шаги

| #   | Действие                                        | Ожидаемый результат                                    |
| --- | ----------------------------------------------- | ------------------------------------------------------ |
| 1   | Войти с валидными credentials                   | Редирект на `/home`, отображается HomePage             |
| 2   | Перейти на вкладку "Кандзи"                     | Отображается страница кандзи                           |
| 3   | Нажать кнопку "+" для добавления кандзи         | Открывается модальное окно "Добавить кандзи"           |
| 4   | Выбрать уровень N5                              | Загружается список доступных кандзи N5                 |
| 5   | Выбрать несколько кандзи (например: 一, 二, 三) | Кандзи отмечены как выбранные                          |
| 6   | Нажать "Добавить"                               | Модальное окно закрывается, кандзи появляются в списке |
| 7   | Перейти на вкладку "Главная"                    | Отображается HomePage                                  |
| 8   | Нажать "Начать урок"                            | Отображается первая карточка с кандзи                  |
| 9   | Нажать "Показать ответ"                         | Отображается ответ и кнопки оценки                     |
| 10  | Нажать "Знаю" (Good)                            | Переход к следующей карточке                           |
| 11  | Повторить для всех карточек                     | После последней — экран "Урок завершён!"               |
| 12  | Проверить статистику на экране завершения       | Отображается количество новых карточек                 |
| 13  | Нажать "На главную"                             | Редирект на `/home`                                    |
| 14  | Проверить, что счётчик кандзи обновился         | Счётчик показывает добавленные кандзи                  |

## User Journey 3: Прохождение урока с разными оценками

**Цель:** Проверить SRS-систему и разные режимы оценки карточек.

### Предусловия

- Пользователь с добавленными кандзи/словами/грамматикой

### Шаги

| #   | Действие                                                  | Ожидаемый результат                |
| --- | --------------------------------------------------------- | ---------------------------------- |
| 1   | Перейти на `/lesson`                                      | Карточки загружены                 |
| 2   | Для первой карточки: "Показать ответ" → "Идеально" (Easy) | Переход к следующей                |
| 3   | Для второй карточки: "Показать ответ" → "Не знаю" (Again) | Переход к следующей                |
| 4   | Для третьей карточки: "Показать ответ" → "Плохо" (Hard)   | Переход к следующей                |
| 5   | Для четвёртой карточки: "Показать ответ" → "Знаю" (Good)  | Переход к следующей или завершение |
| 6   | Завершить все карточки                                    | Экран "Урок завершён!"             |
| 7   | Перейти в профиль → Проверить статистику                  | Статистика обновлена               |

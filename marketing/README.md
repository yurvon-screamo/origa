### Портрет пользователя

Поддержка носителей:

* **1-я волна** Русского и английского языков.  
* **2-я волна** Вьетнамского языка.  
* **3-я волна:** Корейского языка.
* **4-я волна:** Индонезийского языка.
* **5-я волна:** Испанского язык.

**Портреты пользователей:**

1. **Карьера/Переезд**. Цель — сдать JLPT для визы/работы в Японии. Мотивация деньги/переезд.
2. Хобби. Любит аниме, мангу, игры, J-pop. Учит «для себя». Мотивация - геймификация, стрики, красивый дизайн, социальные элементы.

## Поинты

* Обучение на родном языке
* Обучение по контенту юзера
* Фокус на JLPT
* Работа оффлайн
* Проверенный FSRS
* Импорт контента
  * Интеграция контента из других приложений
  * Интеграция контента из учебников
  * Быстрое заведение контента юзера из текстового, фото и аудио материалов
* Простой старт
  * Встроенные словари слов на языках пользователей
  * Встроенные словари кандзи и фуриганы
  * Встроенный перечень правил грамматики
* Комплесное изучение языка
  * Слова
  * Кандзи
  * Грамматика
  * Чтение
  * Аудирование
* Все модели и обработка данных происходят локально на устройстве пользователя, без отправки в облако.
